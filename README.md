# School-Dashboard
as School-Dashboard ist ein Schulprojekt, entwickelt von der BS2AB Gruppe 1 in Kooperation mit der ASC Technologies AG.

Das Ziel des Dashboards ist es, Schülern und Lehrern alle relevanten, zeitkritischen Informationen auf einen Blick bereitzustellen. Die Anwendung dient als zentraler Informationshub, der den aktuellen Stundenplan, Wetterinformationen und weitere wichtige Daten aggregiert darstellt.

## Inhalte

* Hintergrund und Zielsetzung
* Architektur und Kernfunktionen
  * Backend-Technologien (Flask)

## Funktionen

### Flask
Das Backend des School-Dashboards basiert auf dem Python-Webframework Flask. Für die Implementierung von zentralen Funktionen wie Sicherheit, Authentifizierung und E-Mail-Dienste werden folgende Erweiterungen genutzt:

- flask_login
- flask_mail
- flask_wtf
- flask_bcrypt

### Weitere Funktionen:
Zur Abwicklung spezifischer Aufgaben und zur Gewährleistung der Funktionalität werden die folgenden Standardbibliotheken verwendet:

- datetime
- uuid
- secrets
